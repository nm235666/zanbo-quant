## 6. Bug 列表

## Bug 1

- Bug 标题：角色权限策略页中 `limited` / `pro` 区域展示管理员权限项，存在高风险误配可能
- 所属模块：系统管理 / 角色权限策略
- 严重程度：严重
- 优先级：P0
- 前置条件：管理员登录后进入 `/system/permissions`
- 复现步骤：
  1. 登录管理员账号
  2. 打开角色权限策略页
  3. 查看 `pro`、`limited` 角色的权限项列表
- 预期结果：`pro`、`limited` 不应展示 `admin_users`、`admin_system` 或通配权限 `*`
- 实际结果：页面中 `pro`、`limited` 区域同样出现 `admin_users`、`admin_system` 和 `*`
- 影响范围：权限配置、角色隔离、安全边界
- 复现概率：高
- 备注：需人工立即核实这是否仅为展示问题，还是当前真实生效配置

## Bug 2

- Bug 标题：数据源监控页所有监控指标均为 0，未反映当前运行中的系统状态
- 所属模块：系统管理 / 数据源监控
- 严重程度：严重
- 优先级：P1
- 前置条件：管理员登录后进入 `/system/source-monitor`
- 复现步骤：
  1. 登录管理员账号
  2. 打开数据源监控页
  3. 查看数据源、进程、任务编排统计卡片
- 预期结果：监控页应反映真实运行状态，不应在系统有业务数据和实时事件时全部为 0
- 实际结果：数据源正常、进程正常、任务定义、最近运行、成功、失败等核心指标全部显示 0
- 影响范围：运维观测、异常排查、前端管理可用性
- 复现概率：高
- 备注：高度怀疑接口未接通、查询失败被吞掉，或前端未正确消费返回值
