# Zanbo Quant Web 全量重测报告（2026-04-05 UTC）

## 测试目标理解

- 本轮按“新的全量回归”执行，不沿用上一轮通过结论。
- 第一优先级：保障前端展示业务稳定（页面可见、信息正确、交互可用）。

## 风险点

- 异步任务耗时较长，观察窗口内可能无法到终态。
- 高副作用写操作会影响真实数据与任务，不适合本轮直接执行。
- 权限策略页若存在越权权限展示，可能造成误配置风险。

## Bug 列表

## Bug 1

- Bug 标题：角色权限策略页 `pro/limited` 仍出现越权权限项展示迹象
- 所属模块：系统管理 / 角色权限策略
- 严重程度：严重
- 优先级：P0
- 前置条件：管理员登录并进入 `/system/permissions`
- 复现步骤：
  1. 查看 `pro` 与 `limited` 权限列表
- 预期结果：不展示 `*`、`admin_users`、`admin_system`
- 实际结果：本轮自动化检测仍命中越权权限项
- 影响范围：权限配置认知、安全边界
- 复现概率：中
- 备注：需人工复核是否仅展示层异常

## Bug 7

- Bug 标题：多角色分析任务长时间停留中间阶段，终态收敛不稳定
- 所属模块：多角色分析异步链路
- 严重程度：中
- 优先级：P1
- 前置条件：管理员可提交多角色任务
- 复现步骤：
  1. 提交任务
  2. 持续轮询任务状态
- 预期结果：进入终态并产生结果
- 实际结果：长时间停留 `running/research_debate`
- 影响范围：异步任务稳定性、结果可用性
- 复现概率：中
- 备注：需结合后端日志与外部模型耗时排查

## Bug 8

- Bug 标题：日报 Markdown 下载正文缺少所选日期标识
- 所属模块：新闻日报总结 / 下载
- 严重程度：中
- 优先级：P2
- 前置条件：管理员进入 `/intelligence/daily-summaries`
- 复现步骤：
  1. 保持选中日期（本轮样本：`2026-04-05`）

- 影响范围：导出归档与内容追溯一致性
