# 项目使用说明书（摘录）

完整版历史文档见仓库内其它路径；本节仅同步与**导航 UI**相关的当前行为。

## 导航与侧栏

- 顶部为四层 Tab（用户决策 / 数据资产 / 验证研究 / 后台治理）与层级面包屑，详见 [system_overview_cn.md](./system_overview_cn.md) 中「0.1 导航壳层结构」。
- **侧栏按层折叠**：当前 URL 所在层对应的分组会展开显示子页面链接；其它层只显示分组标题，点击标题会进入该层的默认入口页（与顶部 Tab 跳转目标一致）。
- **资讯中心（合并入口）**：侧栏第二层仅保留「资讯中心」→ `/app/data/intelligence`；进入后顶部 Tab 切换国际 / 国内 / 个股 / 日报（按账号权限显示），子路径仍为 `/app/data/intelligence/global-news` 等，旧书签与旧路径重定向不受影响。
