## 四、Bug 清单

### Bug #1: 中文字体显示为方块

| 属性 | 内容 |
|------|------|
| **标题** | 登录页面及全站中文字体显示为方块（□） |
| **模块** | 全局 / 样式 / 字体 |
| **严重程度** | 严重 |
| **优先级** | P1 |
| **状态** | 待修复 |

#### 问题描述
所有中文字符在页面上显示为方块（□），英文内容显示正常。此问题影响全站所有页面，严重影响用户体验。

#### 复现步骤
1. 访问 http://192.168.5.52:8077/login
2. 观察页面文字显示
3. 登录后访问任意页面，问题仍然存在

#### 截图证据
测试过程中截图显示：
- 英文内容（如 "welcome"）显示正常
- 所有中文内容（如 "账号登录"、"用户名"、"密码"）显示为方块

#### 预期结果
中文字体应正常显示，用户能够阅读所有文本内容。

#### 实际结果
所有中文字符显示为方块，用户无法阅读中文内容。

#### 影响范围
- 全站所有页面
- 所有用户（Admin/Pro/Limited）
- 100% 复现概率

#### 可能原因
1. 服务器缺少中文字体文件
2. CSS 字体回退配置不当
3. 字体加载失败或超时

#### 修复建议
1. 在服务器上安装中文字体（如 Noto Sans CJK、Microsoft YaHei）
2. 更新 CSS font-family 配置，添加中文回退字体
3. 考虑使用 Web Font 加载方案
